# Customer Onboarding

## Intake Questions

1. What is the highest-value service you want more customers for?
2. What city, neighborhood, or service area matters most?
3. What action should a visitor take first: call, book, request quote, or visit?
4. Which competitors do you hear customers mention?
5. What customer questions do you answer repeatedly?
6. Are there claims, regulated statements, or promises we must avoid?

## Required Access

Ask only for access needed to fulfill the paid scope:

- Website editor or CMS access if implementing directly
- Google Business Profile manager access if editing profile data
- Existing review request process if improving follow-up

If access is not provided, deliver copy, schema, and instructions for the customer or their webmaster.
